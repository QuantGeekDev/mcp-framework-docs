# MCP Framework Documentation

Documentation site for [mcp-framework](https://github.com/QuantGeekDev/mcp-framework), built with [Fumadocs](https://fumadocs.dev/) and Next.js.

**Live site:** https://mcp-framework.com

## Development

```bash
npm install
npm run dev
```

Open http://localhost:3000.

## Build

```bash
npm run build
```

## Features

- Full-text search via Orama (`/api/search`)
- LLM-friendly endpoints (`/llms.txt`, `/llms-full.txt`)
- Dark mode support
- MDX with syntax highlighting
